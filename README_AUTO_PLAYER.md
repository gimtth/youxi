# 游戏自动玩家使用说明

## 功能说明

这个脚本可以自动玩消除游戏，包括：
- 自动识别游戏网格和图标
- 查找可消除的相邻配对
- 查找需要移动的机会
- 自动执行点击和滑动操作
- 循环直到通关或无解

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 运行自动玩家

```bash
python game_auto_player.py
```

### 2. 操作流程

1. 等待3秒后会截取全屏
2. 点击游戏区域的左上角
3. 点击游戏区域的右下角
4. 脚本会自动识别所有图标并建立网格
5. 按回车键开始自动游戏
6. 脚本会自动执行操作直到通关或无解

### 3. 紧急停止

如果需要紧急停止脚本，将鼠标快速移动到屏幕左上角

## 工作原理

### 1. 网格建立
- 识别所有图标位置
- 通过聚类算法确定行列结构
- 计算每个格子的中心点坐标
- 建立完整的网格数据结构

### 2. 策略优先级

**优先级1：直接消除**
- 扫描所有格子，查找相邻的相同图标
- 直接点击消除

**优先级2：一步移动**
- 查找同类型图标在同一行或同一列
- 计算是否可以通过一次滑动让它们相邻
- 执行滑动操作

**优先级3：多步移动**
- （待实现）需要多次移动的复杂策略

### 3. 操作执行

**点击操作：**
- 使用格子中心坐标
- 避免误触边缘

**滑动操作：**
- 从起始格子中心拖动到目标位置
- 模拟人类操作速度

### 4. 状态更新

每次操作后：
- 重新截取游戏区域
- 重新识别所有图标
- 更新网格状态
- 继续下一轮

## 代码结构

### GameGrid 类
- 存储游戏网格数据
- 管理格子状态
- 提供坐标转换

### GameAutoPlayer 类
- 初始化游戏
- 识别游戏状态
- 查找消除机会
- 执行操作
- 自动游戏循环

## 调试功能

脚本会输出详细信息：
- 网格结构（行数x列数）
- 当前网格状态（可视化）
- 找到的配对数量
- 执行的操作类型
- 操作计数

## 注意事项

1. **游戏窗口要保持可见**：不要被其他窗口遮挡
2. **不要移动鼠标**：脚本运行时鼠标会被自动控制
3. **游戏速度**：如果游戏动画较慢，可能需要调整等待时间
4. **识别准确性**：确保游戏区域选择准确，图标清晰

## 参数调整

在 `game_auto_player.py` 中可以调整：

```python
# 识别阈值
results = self.matcher.match_icons(screenshot, threshold=0.75, use_multiscale=True)

# 操作间隔时间
time.sleep(0.5)  # 消除后等待时间
time.sleep(1)    # 重新识别前等待时间

# 滑动速度
pyautogui.drag(distance, 0, duration=0.5)  # duration控制滑动速度
```

## 已知限制

1. **滑动规则简化**：当前版本对"部分相邻砖块"的滑动规则做了简化，可能需要根据实际游戏调整
2. **只实现了一步移动**：多步移动策略待实现
3. **无回溯搜索**：遇到复杂局面可能无法找到最优解
